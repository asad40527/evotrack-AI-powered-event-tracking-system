# EvoTrack - AI Powered Event Tracking System

EvoTrack is an AI-powered web analytics platform designed to track user interactions and generate intelligent insights from website events.

<img width="748" height="405" alt="image" src="https://github.com/user-attachments/assets/7123fd94-b239-4947-b795-5398e809d57c" />


The system collects user behavior data such as:

- Click events
- Scroll depth
- Session duration
- Form submissions
- Navigation behavior

and applies AI techniques for analysis and automated reporting.
